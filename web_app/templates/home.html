Welcome to our Web App!

Problem Statement

We created this app to help Georgetown students rank their playlists by danceability. The goal is to facilitate social events that focus on dancing and bonding rather than small talk.

We've implemented various attributes for ranking playlists, allowing students to create a playlist to their taste in a time crunch!

Functionality

Navigate to the "Spotify Form" link at the top right to get started.

This app takes in a playlist link and returns the playlist sorted by Spotify's song attribute ratings, ranked in descending order (highest attribute score song at the top and lowest at the bottom).

We utilize Spotify's Web API's. You can find the documentation here. We request track, album, and artist information about songs in a particular playlist.

Important:

  • Must be a playlist, not an album.
  • Use a public playlist.
  • The max playlist length for this code is currently 50 tracks. Providing a URL to a playlist longer than 50 songs will return an error.

FAQ

I'm getting an error!

Check your:

  • Spotify playlist privacy settings
  • Playlist length
  • Internet connection
