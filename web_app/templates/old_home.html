Welcome to our Web App!


Problem Statement

We created this app in hopes of helping Georgetown students rank their playlists by danceability so they can host social events that move away from focusing on people making small talk and towards enabling bonding through dancing.

We also implemented other attributes for ranking besides danceability, so that students can create a playlist to their taste in a time crunch!


Functionality

This app takes in a playlist link and returns the playlist sorted by Spotify's song attribute ratings, ranked in descending order (the highest attribute score song at the top and lowest at the bottom).

Specifically, we will use Spotify's Web API's (documentation found here) request track, album, and artist information about songs in a particular playlist.

Important:

  • Must be a playlist, not an album.
  • Use a public playlist.
  • The max playlist length for this code is currently 50 tracks. Providing a URL to a playlist longer than 50 songs will return an error.

FAQ

I'm getting an error!


Check your:
- Spotify playlist privacy settings
- Playlist length
- Internet connection
- Playlist Link
